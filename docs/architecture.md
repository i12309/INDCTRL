# Архитектура

INDCTRL использует три прикладных сервиса: два FastAPI-сервиса для ESP32 и один
Django-сервис для admin, dashboard и отчетов. Все контейнеры находятся в одной
внутренней Docker-сети, наружу публикуется только Nginx.

Django является источником моделей и миграций БД. FastAPI-сервисы работают с этой же
PostgreSQL-БД напрямую, но не изменяют схему.
