# Улучшить внешний вид Django admin

Подключить готовую тему для Django admin.

Предпочтительный вариант: django-jazzmin, так как он требует минимум изменений и не усложняет проект.

Требования:

- не переписывать admin вручную;
- не создавать собственную админку с нуля;
- сохранить стандартную Django admin;
- сделать внешний вид приятнее и современнее;
- настроить название INDCTRL, заголовки, sidebar и базовые цвета;
- не использовать CDN;
- все static-файлы должны работать offline через collectstatic;
- обновить requirements.txt;
- обновить документацию по web-интерфейсу и запуску;
- все комментарии и настройки — на русском языке;
- не добавлять лишние абстракции.
